Target File:


The target file will be used for all operations.

Backup Folder:

The backup folder will be used to store all saved files.
Use a new empty folder for backups! There is a small chance you can accidentally lose your existing files if you set a backup folder with files already inside.

Snailax & Level Editor Level files:




Miscellaneous


You can set a name for your backups and load from them. (ex: save backup as "abc", load as "abc")
You can technically use this to save multiple level editor levels.
Do not use this option with non UTF-8 files or very large files as it may cause errors when copying the file or cause the backup file to be useless.
